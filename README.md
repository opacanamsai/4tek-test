# Link preview: https://4tek-test-eight.vercel.app/

# How to install & run project

1. **Install dependencies:**
   Make sure you have Node.js installed. Then run:

   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```
